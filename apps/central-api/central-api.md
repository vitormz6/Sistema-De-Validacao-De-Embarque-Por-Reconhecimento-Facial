# Central API

API central do sistema de validação de embarque por reconhecimento facial.

## Responsabilidades

- Cadastro de passageiros
- Cadastro biométrico futuramente
- Cadastro de passagens futuramente
- Recebimento de validações do edge futuramente
- Sincronização com ônibus futuramente
- Auditoria futuramente
